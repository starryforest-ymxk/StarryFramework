# Bezi Unity Plugin

This package contains the Bezi Unity Plugin. It is used to connect to the Bezi App and enables Bezi to search assets, sync project data, make code modifications, and more.

Learn more about Bezi in our documentation: https://docs.bezi.com

## Security and Data Privacy

Because Bezi connects to the project, we invest heavily in IP security to keep your project data safe. No project data is ever used to train models. We encourage you to review our security program and security FAQs at: https://docs.bezi.com/bezi/security
